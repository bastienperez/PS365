id: Invoke-M365AppsDownload title: Invoke-M365AppsDownload hide_title: false hide_table_of_contents: false ---  ## SYNOPSIS  Downloads the Microsoft 365 Apps binary using the Office Deployment Tool (ODT).  ## SYNTAX  ```powershell Invoke-M365AppsDownload [-ConfigFilePath] <String> [-ProgressAction <ActionPreference>] [<CommonParameters>] ```  ## DESCRIPTION  This script downloads the Office Deployment Tool from Microsoft's official site, extracts it, and uses a specified configuration XML file to download the Microsoft 365 Apps binary.  ## EXAMPLES  ### EXAMPLE 1  ```powershell Invoke-M365AppsDownload -ConfigFilePath .\Configuration-XX.xml ```  Runs the script to download the Microsoft 365 Apps binary.  ## PARAMETERS  ### -ConfigFilePath  The path to the configuration XML file used for the download. You can generate this file using the Office Customization Tool (OCT) at https://config.office.com/. Can be a relative or absolute path.  ```yaml Type: String Parameter Sets: (All) Aliases:  Required: True Position: 1 Default value: None Accept pipeline input: False Accept wildcard characters: False ```
