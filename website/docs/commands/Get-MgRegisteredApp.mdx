id: Get-MgRegisteredApp title: Get-MgRegisteredApp hide_title: false hide_table_of_contents: false ---  ## SYNOPSIS  Get registered application in Microsoft Graph  ## SYNTAX  ```powershell Get-MgRegisteredApp [[-ApplicationID] <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>] ```  ## DESCRIPTION  Retrieves registered applications from Microsoft Graph. If no ApplicationID is provided, returns all applications with selected properties.  ## EXAMPLES  ### EXAMPLE 1  ```powershell Get-MgRegisteredApp -ApplicationID "your-application-id" ```  This command retrieves the registered application with the specified Application ID.  ### EXAMPLE 2  ```powershell Get-MgRegisteredApp ```  This command retrieves all registered applications.  ## PARAMETERS  ### -ApplicationID  The unique identifier of the registered application to retrieve. If not provided, all applications are returned.  ```yaml Type: String Parameter Sets: (All) Aliases:  Required: False Position: 1 Default value: None Accept pipeline input: False Accept wildcard characters: False ```
