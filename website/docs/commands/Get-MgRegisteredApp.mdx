## SYNOPSIS

Get registered application in Microsoft Graph

## SYNTAX

```powershell
Get-MgRegisteredApp [[-ApplicationID] <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Retrieves registered applications from Microsoft Graph.
If no ApplicationID is provided, returns all applications with selected properties.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-MgRegisteredApp -ApplicationID "your-application-id"
```

This command retrieves the registered application with the specified Application ID.

### EXAMPLE 2

```powershell
Get-MgRegisteredApp
```

This command retrieves all registered applications.

## PARAMETERS

### -ApplicationID

The unique identifier of the registered application to retrieve.
If not provided, all applications are returned.
