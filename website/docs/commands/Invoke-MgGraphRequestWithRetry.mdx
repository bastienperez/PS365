## SYNOPSIS

Wraps Invoke-MgGraphRequest with automatic retry on HTTP 429 (throttling).

## SYNTAX

```powershell
Invoke-MgGraphRequestWithRetry [-Uri] <String> [[-Method] <String>] [[-Body] <Object>] [[-MaxRetries] <Int32>]
 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Manual pagination loops over large tenants can be throttled by Graph.
Without
a retry, the catch block typically logs a warning and continues, silently
returning incomplete results.
This helper retries on 429, honoring the
Retry-After header when present and otherwise backing off exponentially
(capped at 60s).
Any non-429 error is rethrown unchanged.
